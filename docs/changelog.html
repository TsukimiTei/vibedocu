Changelog

版本更新记录

v1.14.0 Latest 2026-03-17

Added

  • 4 个新淡雅主题 — Sage、Ocean Mist、Dusty Rose、Lavender
  • 多窗口项目目录隔离 — 每个窗口独立维护项目目录，互不干扰
  • 关闭窗口保护 — 自动保存 + 二次确认弹窗
  • Open/New 两步弹窗 — 选择文件后选择项目目录，流程更清晰
  • Settings 分栏布局 — 左侧 Tab 导航（AI 配置 / 工作空间 / 智能代理 / 外观）
  • Welcome 页面重设计 — 卡片网格 + 工具栏布局
  • MCP 分析提速 9x — 预取所有上下文嵌入 prompt，27 turns → 2-3 turns

Fixed

  • 修复多窗口 projectDir 跨窗口污染（persist merge + 原子 read-merge-write）
  • 修复目录选择器每次打开默认目录相同的问题（传入 defaultPath）
  • 修复连续分析报 "Analysis already in progress" 的竞态条件
  • 修复 Token 显示中间态闪烁 0.0k 的问题
v1.11.0 2026-03

Added

  • Claude Code 模式 — 新增 MCP（Model Context Protocol）AI 分析模式，可在设置中与 OpenRouter 模式切换
  • MCP Server — 7 个工具供 Claude Code 直接读写文档、扫描项目文件、保存分析结果
  • Pipeline 进度 UI — 首次分析展示 6 步全流程，增量分析展示 3 步流程，实时滚动进度
  • Session 复用 — MCP 模式下通过 --resume 复用会话，增量分析速度显著提升
  • 解释问题 / 解释选项 — 两种 AI 模式均支持逐题解释和逐选项 AI 解释

Fixed

  • 修复分析错误信息被 setLoading 吞掉的问题
  • 修复 Pipeline 步骤顺序错乱（pending 步骤出现在 running 上方）
  • 修复完成信号竞争导致 stats 显示 0.0s / 0 turns
v1.1.0 2026-03

Added

  • Obsidian 同步 — 支持选择 Obsidian vault 目录自动同步文档
  • 多页面系统 — Tab 栏管理多个文档页面，支持版本迭代
  • 页面标题显示 — 编辑器顶部展示当前页面标题
  • 问题折叠 — 已回答的问题可折叠收起，界面更清爽
  • Context 上下文系统 — 支持为 AI Agent 提供项目上下文信息

Changed

  • Copy Message 改为输出文件路径而非内联内容，减少 token 消耗
  • 新页面自动添加 h2 提示，引导用户开始书写
  • 页面系统重设计为可滚动多编辑器视图 + 自动保存
v1.0.0 2026-02

Added

  • 核心编辑器 — 基于 Milkdown 的 WYSIWYG Markdown 编辑器
  • AI Agent 提问 — 从 8 个维度分析文档，生成针对性问题
  • 完成度评分 — 实时显示各维度独立评分和总分
  • Copy Message — 一键生成完整 prompt 交给 Coding Agent
  • 300+ 模型支持 — 通过 OpenRouter 接入 Claude、GPT、Gemini 等
  • 深色 / 暖光主题切换
  • 新手引导对话框和使用手册
  • Cmd+Enter 快捷键发送
  • JSON 结构化输出回退解析 — 提升 AI 响应稳定性
  • 官网 Landing Page + macOS 安装指南
