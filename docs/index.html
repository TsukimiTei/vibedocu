# 产品需求文档 — 个人财务管理 App

## 问题陈述
年轻上班族缺乏直观的工具追踪日常开支，
导致月底经常超支却不知道钱花在哪里。

## 目标用户
22-35 岁一线城市上班族，月收入 8k-25k，
有记账意识但现有工具太复杂。

## 核心功能
1. 快速记账（< 3 秒完成一笔）
2. 智能分类（AI 自动识别消费类别）
3. 月度预算 + 超支提醒
4. 可视化报表（周/月/年维度）
